Многостраничный сайт на JavaScript

Современный многостраничный сайт с красивым дизайном и интерактивными элементами. Создан на чистом JavaScript без использования фреймворков.

Особенности

- 5 полноценных страниц с уникальным контентом
- Адаптивный дизайн - корректно отображается на всех устройствах
- Яркая цветовая схема - лаймовый фон с красными акцентами
- Плавные анимации переходов между страницами
- Интерактивные элементы - кнопки, формы, карточки
- Профессиональная визитка с контактной информацией

Страницы сайта

1. Главная (`home`)
- Герой-секция с приветствием
- Блок преимуществ компании
- Призыв к действию

2. О нас (`about`)
- Информация о компании
- Презентация команды
- Опыт работы и специализация

3. Услуги (`services`)
- 6 основных услуг компании
- Веб-разработка, мобильные приложения, дизайн
- SEO, аналитика, облачные решения

4. Портфолио (`portfolio`)
- Примеры выполненных проектов
- Используемые технологии
- Демонстрация компетенций

5. Контакты (`contact`)
- Профессиональная визитка
- Форма обратной связи
- Реквизиты компании
- Социальные сети

Дизайн

Цветовая схема:
- Фон: Лаймовый (`#e0f7e0`) - мягкий и приятный для глаз
- Акценты: Красный (`#dc2626`) - кнопки, иконки, градиенты
- Текст: Темно-серый (`#333`) - оптимальная читаемость
- Навигация: Темный (`#1a1a1a`) - контрастный и стильный

Элементы интерфейса:
- Карточки с тенями и hover-эффектами
- Плавные переходы между страницами
- Интерактивные кнопки с анимациями
- Адаптивное мобильное меню

Техническая реализация

Архитектура:
- Чистый JavaScript** (ES6+)
- CSS3 с Flexbox/Grid
- HTML5 семантическая разметка
- Single Page Application подход

Ключевые функции:
```javascript
class Router {
    // Навигация между страницами
    // Динамическая загрузка контента
    // Обновление активных ссылок
}
```

Адаптивность:
- Mobile-first подход
- Гамбургер-меню для мобильных устройств
- Гибкие сетки (CSS Grid)
- Медиа-запросы для разных разрешений

Запуск проекта

Локальная разработка:
```bash
 Перейдите в папку проекта
cd project-folder

 Запустите локальный сервер
python -m http.server 8000

 Откройте в браузере
http://localhost:8000
```

Альтернативные способы запуска:
```bash
 С помощью Node.js
npx serve .

 С помощью PHP
php -S localhost:8000
```

Контакты и визитка

Кастомизация

Легко изменить:
- Цветовую схему в `style.css`
- Контент страниц в `script.js`
- Логотип и брендинг
- Количество и состав страниц

Добавить новые страницы:
1. Добавить ссылку в навигацию
2. Создать метод в классе Router
3. Реализовать HTML-контент
